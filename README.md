## 部署方式

本项目使用 GitHub Pages 进行静态部署：

1. Fork 或克隆本仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择分支（通常为 `main` 或 `master`）
4. 网站将自动部署到 `https://yourusername.github.io/AIwork`
